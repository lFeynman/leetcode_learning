## 6.对角线遍历

### 原题链接:https://leetcode.cn/leetbook/read/array-and-string/cuxq3/

**题目描述:**

```
给你一个大小为 m x n 的矩阵 mat ，请以对角线遍历的顺序，用一个数组返回这个矩阵中的所有元素。
```

**思路:**

```
画图,观察到i+j=k恒成立,分别找奇数和偶数的规律
```

**代码:**

```python
class Solution:
    def findDiagonalOrder(self, mat: List[List[int]]) -> List[int]:
        n = len(mat) 
        m = len(mat[0]) if n > 0 else 0 
        res = []
        total_diagonals = n + m - 1 
        
        for k in range(total_diagonals):
            if k % 2 == 0:
                # 偶数层
                i = min(k, n - 1) #防止超出
                j = k - i         
                while i >= 0 and j < m: 
                    res.append(mat[i][j])
                    i -= 1
                    j += 1
            else:
                # 奇数层
                j = min(k, m - 1)  
                i = k - j         
                while i < n and j >= 0:  
                    res.append(mat[i][j])
                    i += 1
                    j -= 1
        return res
```
