## 7.最长公共前缀

### 原题链接:https://leetcode.cn/leetbook/read/array-and-string/ceda1/

**题目描述:**

```
编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 ""。
```

**思路:**

```
纵向遍历,利用第一个字符串去同时匹配其他的字符串的前几个字母
```

**代码:**

```python
class Solution:
    def longestCommonPrefix(self, strs: List[str]) -> str:
        #处理空字符串
        if not str:
            return ""
        #纵向遍历
        prefix=strs[0]
        for i in range(len(prefix)):
            current_char=prefix[i]
            for s in strs[1:]:
                if i>=len(s) or s[i]!=current_char:
                    return prefix[:i]#返回已匹配好的字符串
        return prefix#全部匹配
```
