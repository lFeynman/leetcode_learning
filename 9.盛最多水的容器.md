## 9.盛最多水的容器

### 原题链接:https://leetcode.cn/problems/container-with-most-water/solutions/207215/sheng-zui-duo-shui-de-rong-qi-by-leetcode-solution/?envType=problem-list-v2&envId=array

**题目描述:**

```
给定一个长度为 n 的整数数组 height 。有 n 条垂线，第 i 条线的两个端点是 (i, 0) 和 (i, height[i]) 。
找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。
返回容器可以储存的最大水量。
说明：你不能倾斜容器。
```

**思路:**

```
采用双指针算法:分别从两端开始,储水体积为min(h[x],h[y])*(y-x),移动矮的那个,因为矮的那个限制了最大值,比如左端为较小的那个,如果移动右侧则min(x,y1)<=min(x,y),且距离减少,故较小的那个限制了当前最大的体积
```

**代码:**

```python
class Solution:
    def maxArea(self, height: List[int]) -> int:
        i, j, res = 0, len(height) - 1, 0
        while i < j:
            if height[i] < height[j]:
                res = max(res, height[i] * (j - i))
                i += 1
            else:
                res = max(res, height[j] * (j - i))
                j -= 1
        return res
```
